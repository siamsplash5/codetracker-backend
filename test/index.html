問題文

A A A 駅と B B B 駅を結ぶ鉄道があり、運賃は X X X 円です。

また、 B B B 駅と C C C 駅を結ぶバスがあり、運賃は Y Y Y 円です。

joisinoお姉ちゃんは、 A A A 駅から B B B 駅まで鉄道で移動し、 B B B 駅から C C C 駅までバスで移動すると、バスの運賃が半額になる特別券を手に入れました。

この特別券を用いたとき、 A A A 駅から C C C 駅まで移動するのにいくらかかるか求めてください。

制約

  • 1 ≤ X , Y ≤ 100 1 \leq X,Y \leq 100 1≤X,Y≤100
  • Y Y Y は偶数
  • 入力は全て整数

入力

入力は以下の形式で標準入力から与えられる。

XXX YYY

出力

A A A 駅から C C C 駅まで移動するのに x x x 円かかるとき、 x x x を出力せよ。

入力例 1 Copy

Copy 
81 58

出力例 1 Copy

Copy 
110
  • 鉄道の運賃は 81 81 81 円です。
  • バスの運賃は半額となるため、 58 58 58 ⁄ 2 = 29 2=29 2=29 円です。

よって、 A A A 駅から C C C 駅まで移動するのに 110 110 110 円かかります。

Problem Statement

There is a train going from Station A A A to Station B B B that costs X X X yen (the currency of Japan).

Also, there is a bus going from Station B B B to Station C C C that costs Y Y Y yen.

Joisino got a special ticket. With this ticket, she can take the bus for half the fare if she travels from Station A A A to Station B B B by train and then travels from Station B B B to Station C C C by bus.

How much does it cost to travel from Station A A A to Station C C C if she uses this ticket?

Constraints

  • 1 ≤ X , Y ≤ 100 1 \leq X,Y \leq 100 1≤X,Y≤100
  • Y Y Y is an even number.
  • All values in input are integers.

Input

Input is given from Standard Input in the following format:

XXX YYY

Output

If it costs x x x yen to travel from Station A A A to Station C C C, print x x x.

Sample Input 1 Copy

Copy 
81 58

Sample Output 1 Copy

Copy 
110
  • The train fare is 81 81 81 yen.
  • The train fare is 58 58 58 ⁄ 2 = 29 2=29 2=29 yen with the 50 50 50% discount.

Thus, it costs 110 110 110 yen to travel from Station A A A to Station C C C.
