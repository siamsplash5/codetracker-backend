Problem Statement

Takahashi wants to gain muscle, and decides to work out at AtCoder Gym.

The exercise machine at the gym has N N N buttons, and exactly one of the buttons is lighten up. These buttons are numbered 1 1 1 through N N N. When Button i i i is lighten up and you press it, the light is turned off, and then Button a i a_i ai​ will be lighten up. It is possible that i = a i i=a_i i=ai​. When Button i i i is not lighten up, nothing will happen by pressing it.

Initially, Button 1 1 1 is lighten up. Takahashi wants to quit pressing buttons when Button 2 2 2 is lighten up.

Determine whether this is possible. If the answer is positive, find the minimum number of times he needs to press buttons.

Constraints

  • 2 ≤ N ≤ 1 0 5 2 ≤ N ≤ 10^5 2≤N≤105
  • 1 ≤ a i ≤ N 1 ≤ a_i ≤ N 1≤ai​≤N

Input

Input is given from Standard Input in the following format:

:

Output
